### (b) Diode Current

Formula:
ID = IS (e^(VD / nVT) - 1)

Given:
IS = 40 nA = 40 × 10⁻⁹ A  
VD = 0.5 V  
n = 2  
VT = 0.026 V  

Calculation:

ID = 40 × 10⁻⁹ (e^(0.5 / (2 × 0.026)) - 1)

ID = 40 × 10⁻⁹ (e^9.62 - 1)

e^9.62 ≈ 15000

ID ≈ 40 × 10⁻⁹ × 15000  
ID ≈ 0.0006 A  

Final Answer:
ID ≈ 0.6 mA
